# ✦ GemSoul — เว็บควิซอัญมณี

เว็บไซต์ทายนิสัยผ่านอัญมณี พร้อมคลังความรู้เกี่ยวกับอัญมณีแต่ละชนิด

## โครงสร้างไฟล์

```
gem-quiz/
├── index.html        ← หน้าแรก (Homepage)
├── quiz.html         ← หน้าแบบทดสอบ
├── gems.html         ← คลังอัญมณี (Encyclopedia)
├── css/
│   └── style.css     ← สไตล์ทั้งหมด
├── js/
│   ├── starfield.js  ← แอนิเมชันดาวพื้นหลัง
│   ├── nav.js        ← เมนูนำทาง
│   ├── quiz.js       ← ลอจิกแบบทดสอบ
│   └── gems.js       ← ฟิลเตอร์หน้าคลังอัญมณี
└── README.md
```

## อัญมณีทั้งหมด 8 ชนิด
- 🔴 ทับทิม (Ruby) — กล้าหาญ ผู้นำ
- 🔵 แซฟไฟร์ (Sapphire) — ฉลาด ปัญญา
- 💚 มรกต (Emerald) — เมตตา รักสงบ
- 💜 อเมทิสต์ (Amethyst) — สร้างสรรค์ อิสระ
- 💎 เพชร (Diamond) — สมบูรณ์แบบ มุ่งมั่น
- 🟡 ซิทรีน (Citrine) — ร่าเริง สดใส
- ⚫ ออบซิเดียน (Obsidian) — ลึกลับ แข็งแกร่ง
- 🌙 มูนสโตน (Moonstone) — ละเอียดอ่อน สัญชาตญาณ

## วิธีใช้งาน

### เปิดบน Browser โดยตรง
เพียงดับเบิลคลิกที่ `index.html` เพื่อเปิดใช้งาน

### อัพโหลดไปยัง Wix
1. ไปที่ Wix Editor
2. ใช้ Wix Velo (Dev Mode) หรือ Wix HTML iFrame
3. อัพโหลดไฟล์ทั้งหมดผ่าน Wix Media Manager
4. หรือคัดลอก HTML ไปวางใน Custom HTML Element

### Host บน GitHub Pages / Netlify
1. สร้าง repository บน GitHub
2. อัพโหลดโฟลเดอร์ทั้งหมด
3. เปิด GitHub Pages หรือเชื่อมกับ Netlify
4. เว็บจะใช้งานได้ทันที (ฟรี!)
